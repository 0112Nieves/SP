- Used language：C++
- Compile way：
把makefile跟assembler.c++一起放到同一資料夾，打開linux(Ubuntu)，在終端機找到該資料夾的位置後下make，compile過後執行，輸入有測資的file name後按enter(需在同一資料夾有該file)，會產生兩個txt檔，一個是optab另一個是object code。
